Welcome to all things Bootstrap!

Bootstrap is a framework that was created to help keep consistency and ease while creating responsive mobile frist webites. It was created by Mark Otto and Jacob Thornton at Twitter as a way to reduce the amount of inconsistencies within layouts and create a "one stop shop" for anything coding related! The beauty of this framework is that it is not only open source for anyone to access, but it has plenty of instuctions and explinations on how to use each component of bootstrap and has plenty of guides to help new users. All in all, Bootstrap is AMAZING!!

Some of the Perks

Moblie First

Bootstrap is a framework desinged to aid in creating mobile first responsive layouts. This means that while adding broostrap components and elements into ones HTML we are able to manipulate these compenents adn elements due to a flex-box grid and a 12 column system. These systems have a number of classes that aid in creating the ideal layout for every webiste.

Pre-coded CSS and Javascript

Bootsrap has a very powerful amount of pre-coded CSS and Javascrpit that is added to any code when adding any HTML component to ones code. By using specific classes we can activate certain other CSS and Javascript propoerties that can add different visual apeal to to any cde that will aid in creating layouts of all kinds!

Layouts

Bootstrap has a build in layout system that has a number of elemets such as gutter width, breakpoints, columns, rows, etc. These all aid in the creation of seamless layouts in order to get the desiered design required. These can be activated by useing very specific classes with some added number to make minor specifications of certain elements.

Helper options

These options have been added to aid in more designing creative layouts with using and help by giving instruction on each seperate element. Each of these have a specific class that is used to activate them as well as other classes can be used to at styling to each element.

Bootstrap components are a variety of elementsthat can be added to you HTML layouts to add more visul interest and aid is a variety of ways.
